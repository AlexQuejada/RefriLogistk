Sistema de gestión (ERP ligero) para técnicos y negocios de servicio técnico.

Permite administrar clientes, registrar órdenes de servicio, programar trabajos en calendario y visualizar métricas del negocio desde un dashboard.

Funcionalidades

Gestión de clientes (crear, editar, eliminar)
Historial de servicios por cliente
Creación de órdenes de servicio
Dashboard con métricas del negocio
Calendario para programar servicios
Visualización de servicios programados
Interfaz estilo ERP moderna

Tecnologías utilizadas

PHP (arquitectura MVC personalizada)
MySQL
Bootstrap 5
JavaScript
FullCalendar
FontAwesome

Objetivo del proyecto

Este proyecto fue desarrollado como sistema real para la gestión de un negocio de servicio técnico, con el objetivo de mejorar el control de clientes, seguimiento de servicios y organización del trabajo.
